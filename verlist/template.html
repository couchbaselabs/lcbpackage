Couchbase C Client Archives

This contains a full listing of release versions of the C client library (libcouchbase). The most current package and installation information can be found at http://couchbase.com/communities/c

Using

This page contains links to all versions of libcouchbase. For Debian and Redhat-based systems these files are present as native archives wrapped inside tar archives. Thus in such a scenario, one would do: 
# tar xf libcouchbase-2.2.0-centos62_x86_64.tar
# cd libcouchbase-2.2.0-centos62_x86_64.tar
# rpm -ivh ...
It may also be necessary to remove your existing packages, otherwise some package managers will complain about downgrading.
